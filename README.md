# Algorithms and Data Structures in Python

This repository contains my personal studies, exercises, and implementations of algorithms and data structures using Python. It serves as a learning log and a reference guide for computer science concepts.

## Prerequisites

To run the scripts in this repository, you need to have Python installed on your machine.

- Python 3.x

## How to Run

Clone this repository and run the individual scripts using the terminal.

```bash
git clone [https://github.com/Edu-2de/algorithms-data-structures.gitt](https://github.com/Edu-2de/algorithms-data-structures.git)
cd algorithms-data-structures
python path/to/specific_script.py
```
